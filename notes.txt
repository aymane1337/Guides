find -type f \( -name '*~' -o -name '#*#' \) -print -delete

find : The find command is used to search for files and directories in a directory hierarchy.
-type f : restricts the search to regular files only (not directories, symbolic links, etc.).
\( -name '*~' -o -name '#*#' \): 
\( and \): These parentheses are used to group the conditions inside them
-name '*~': This part looks for files whose names end with ~
-o: This is the logical OR operator.
-name '#*#': This part looks for files whose names start and end with #
-print : will print them
-delete : will delete them
